# Password Generator

Secure password generator.

Generator password aman.


## Description

Creates cryptographically secure passwords with customizable length and character sets.

## Deskripsi

Membuat password aman secara kriptografis dengan panjang dan set karakter yang dapat disesuaikan.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Generate: python passgen.py
- Custom length: python passgen.py -l 20 --symbols
- Demo: python passgen.py --demo

## Penggunaan

- Generate: python passgen.py
- Panjang kustom: python passgen.py -l 20 --symbols
- Demo: python passgen.py --demo
